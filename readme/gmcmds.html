GM Commands

Previous Next

Explanation

GM Commands are commands that can be typed in by players and give a certain effect. The commands a player can use depend on their user level (not their base or job level, but a level set by the administrator of the server). The higher a player's user level, the more commands he or she may use. The default level for each command can be changed by the administrator.

All GM commands begin with a certain character, which is set by the administrator. By default, this character is @.
Administrator sets special character to replace the begining of remote GM commands (begining by @char). By default, this character is #.
Example: #rura geffen 75 75 <char name|account_id> is same @charrura geffen 75 75 <char name|account_id>.

The following is the syntax for GM commands:
@command [ < param_type_choice1 | param_type_choice2 > ]

Command Explanations

Help:

@h/@help [<page_number>|<keyword>]
Displays the help.txt file, a short help of GM commands.
If a keyword is given:
this command display only help of this keyword (if key word is found).
this command display all keywords (if keyword is not found).
special keywords: 'key' or 'keyword' to display all keywords too.
If a page is given:
1 page of 50 lines id displayed, depends of the page number you want.
If nothing is given, hall help file is displayed.
