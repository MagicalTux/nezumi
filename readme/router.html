Router guide

Previous Next

Note: This guide is inspired from a post found on the Freya's forum and created by Limonada.

Your ip's (for the example):

Lan ip/mask: 10.0.0.0/255.255.255.0
The gateway: 10.0.0.138
Lan ip of your server: 10.0.0.1
Wan IP of your server: 201.0.7.23

Note: This guide is based on the fact that all servers (login, char and map) are installed on the same computer.

Let's start

Forward the ports (here default ports): 6900, 5121 and 6121 to your Lan IP: 10.0.0.1.
Do it using your router config (usually NAT).

Configure your login server

Open your conf/login_athena.conf.

Set configuration like this:

// put here the LAN IP of your char-server
lan_char_ip: 10.0.0.1

// put here the Subnet mask of your LAN
subnet: 10.0.0.0
subnetmask: 255.255.255.0

Save the file.

Configure your char server

Open your conf/char_athena.conf.

Set configuration like this:

// Login Server IP
// You should only need to set this if you are running behind a
// firewall or on a machine with multiple interfaces. In that case,
// you need to specify the IP address you wish to export to the entire world.
// You must specify the IP adress if your login-server is not on the same computer of your char-server.
//login_ip:127.0.0.1 <-- here, leave the comment or (better) set to
login_ip:10.0.0.1

// Character Server IP
// You should only need to set this if you are running behind a
// firewall or on a machine with multiple interfaces. In that case,
// you need to specify the IP address you wish to export to the entire world.
char_ip:201.0.7.23

// put here the LAN IP of your map-server
lan_map_ip: 10.0.0.1

// put here the Subnet mask of your LAN
subnet: 10.0.0.0
subnetmask: 255.255.255.0

Save the file.

Configure your map server

Open your conf/map_athena.conf.

Set configuration like this:

// Character Server IP
// You should only need to set this if you are running behind a
// firewall or on a machine with multiple interfaces. In that case,
// you need to specify the IP address you wish to export to the entire world.
// You must specify the IP adress if your char-server is not on the same computer of your map-server.
//char_ip:127.0.0.1 <-- here, leave the comment or (better) set to
char_ip:10.0.0.1

// Map Server IP
// You should only need to set this if you are running behind a
// firewall or on a machine with multiple interfaces. In that case,
// you need to specify the IP address you wish to export to the entire world.
map_ip:201.0.7.23

Save the file.

To finish

Configure your sclientinfo.xml like this:
If you connect from your lan, use: 10.0.0.1.
If you connect from the wan, use: 201.0.7.23.
